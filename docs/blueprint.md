# **App Name**: EduAI Tutor

## Core Features:

- AI Tutor: AI-powered tutor that answers questions related to the GCE Advanced Level syllabus for Biology, Combined Maths, Physics, and Chemistry in both Sinhala and English. The tutor is designed to be a tool for students, so that the LLM is not overused, but the student has control over its reasoning to see if they wish to follow that line of reasoning.
- Language Selection: A multi-language selector, which can allow the student to formulate and read the response in the language of their choice
- Chat Interface: Simple chat interface for easy interaction with the AI tutor.
- File Uploads: The application allows file uploads, so students can share homework problems or scans from their textbook to help clarify their request.

## Style Guidelines:

- Primary color: Light blue (#E0F7FA) for a calm and studious atmosphere.
- Secondary color: Soft grey (#F5F5F5) for backgrounds to ensure readability.
- Accent: Teal (#008080) for highlights and interactive elements.
- Clear, sans-serif fonts for readability.
- Simple, professional icons for navigation and features.
- Clean, organized layout with a focus on easy navigation.
- Subtle transitions for a smooth user experience.
